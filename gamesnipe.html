Laungauges and frameworks used

Get up to date and don't get to miss all the latest games from all platforms that get released every month. To all gaming enthusiasts Gamesnipe is a mobile gaming magazine app that includes game reviews, ratings, screenshots, platforms and descriptions about the games. With over 500,000 games powered by rawg api gamesnipe covers many original games, mods, DLC and updated games with a rich user friendly UI/Ux design making it easier to use and experiance a real magazine like features within the app through page flipping effects for navigating all games and detailed sections for reading about the game.

Gamesnipe uses the rawg api from RAWG a video game discovery platform and community to get most of its gaming data

When you install gamesnipe it comes with a free api key it uses the rawg api from RAWG a video game discovery platform and community to get most of its gaming data visit rawg for more info. This key is shared by default to all users who use this app hence it's limit runs out if more people use it and you will have to wait until next month to reuse it again

Gamesnipe or mwatech does not own the rawg database hence why a public api key given to us by rawg is used to access it's gaming database for free and it's fairly easy you can get your own personal rawg api key as well for free without sharing limitations instead of relying on the default and limited one that comes with gamesnipe. So to fix this simple issue head over to rawgapi to create your own api key. Once inside the website click the get api key button then sign in with your email to create an account with rawg to get your key(if you cant think of any, use the words gamesnipe user as a purpose for acquiring the key) once your api key is created which is a long string of characters eg 5d7g367mt6... copy that string and paste it on the set personal key text field on your gamesnipe app
